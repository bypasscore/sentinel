# Security Policy

## Purpose

Sentinel is a **security research framework** designed for studying anti-cheat
detection mechanisms and understanding their internals. It is intended for:

- Security researchers analyzing game protection systems
- Anti-cheat developers testing their detection coverage
- Academic research on software protection techniques
- Penetration testers with proper authorization

## Responsible Disclosure

If you discover a vulnerability in an anti-cheat system using this framework,
please follow responsible disclosure practices:

1. **Do not** use findings to create cheats or exploits for live games
2. Report vulnerabilities to the anti-cheat vendor through their official channels
3. Allow reasonable time for fixes before public disclosure
4. Do not publish working exploits without vendor coordination
